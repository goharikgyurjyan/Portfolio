Projet Java - Trains

Le projet Trains est un projet qui s'inscrit dans le cadre des SAE (Situations d'Apprentissage et d'Évaluation). Il a été découpé en 3 phases distinctes réparties entre mars 2024 et juin 2024. Le projet dans sa globalité consiste à implémenter le jeu de plateau "Trains".

Il y a d'abord eu une première phase de développement de la mécanique du jeu en Java, où mon bînome et moi avons dû compléter les squelettes de codes fournis, ajouter des classes qui nous semblaient nécessaires, afin de correspondre avec les règles du jeu et la mécanique attendue.

Ensuite, il y a eu une phase de mise en oeuvre des algorithmes de graphes. Bien que pas vraiment intégré au projet, il s'agissait de coder divers algorithmes de graphes afin d'optimiser certains algorithmes déjà présents dans la phase 1.

Enfin la dernière phase consistait à utiliser JavaFX qui est un framework permettant créer une interface graphique afin de modéliser le plateau du jeu Trains. Pour ce faire, nous disposions de la correction de la phase 1 pour nous concentrer sur l'affichage.

Concernant la répartition du travail, elle a été équitable durant toutes les phases du projet, une moitié du code a été faite par Corentin Cano, mon binôme, et l'autre par moi.

Compétences sollicitées

  • Travail d'équipe : Planification du projet, réunions régulières
  • Git : Versionning du site (historique des versions), collaboration en équipe
  • Java : Fonctionnement global du projet, implémentation du jeu de société au travers des différentes classes
  • JavaFX : Interface graphique du projet
  • Scene Builder : Outil de conception d'interface graphique
