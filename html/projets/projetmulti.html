Projet Multi - Break The Code

Le projet Break The Code est un projet qui s'inscrit dans le cadre des SAE (Situations d'Apprentissage et d'Évaluation). Le projet s'est déroulé durant tout le premier semestre de la deuxième année, de septembre 2024 à janvier 2025.

C'est avec 4 de mes camarades que nous avons consitué une équipe de développeurs prêts à accompagner notre client fictif de l'association Yggame.

Le principe de ce projet paraissait à première vue simple, fournir à ladite association, une plateforme permettant d'organiser ces CTF (Capture The Flag). Les Capture The Flag sont des défis de cybersécurité déstinés à faire découvrir la cybersécurité à un public qui n'est pas familier avec ce domaine.

Pour l'organisation du projet, afin de respecter les principes agiles vus en cours, il a fallu nommer un product owner ainsi qu'un scrum master. Ces derniers se chargeaient du respect des principes agiles notamment par des réunions journalières, appelées Daily Meetings. Le projet s'organisait en plusieurs sprints, à l'issue desquels un point était fait avec le client avant de faire la livraison finale du produit.

L'outil principal pour ce projet a été le framework CTFd, j'ai donc contribué en ajoutant divers plugins à la demande du client ou par mon initiative personnelle, dans le but d'améliorer l'expérience utilisateur, pour ce même but j'ai également pu créer des thèmes de couleur à l'aide de ma collègue Manon Lopes.

Compétences sollicitées

  • Travail d'équipe : Planification du projet, réunions régulières
  • Gestion de projet : Méthodes Agile
  • Git : Versionning du site (historique des versions), collaboration en équipe
  • Python (Flask) : Langage principal pour le back-end de CTFd, j'ai pu le manipuler lors de la création de certains plugins.
  • JavaScript : Utilsé dans certaines pages pour récupérer du contenu ou modifier les agissements de la page, j'ai également pu le manipuler lors de la création de certains plugins.
  • HTML : Structuration des pages
  • CSS : Style des pages
  • SQL : Utilisation du SQL dans certains défis du CTF
  • PHP : Utilisation du PHP dans certains défis du CTF
