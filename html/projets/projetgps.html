Projet Externe - GoPetSit™

Le projet GoPetSit™ est un projet externe aux projets scolaires. Il a débuté en Janvier et est toujours en cours.

C'est un projet qui est né à la suite l'appel à participation d'une de mes camarades de classe, Héloise Rigaux. J'ai donc rejoint une petite équipe de développeurs de l'IUT de Montpellier-Sète afin de répondre aux besoins de Lucie Sandré , fondatrice de GoPetSit™.

Dans ce projet, j'ai notamment pris part à la conception et évolution de la base de données. J'ai également pu développer un système de messagerie instantanée entre les utilisateurs, une fonctionnalité clé au sein du site.

Dans un soucis de confidentialité, comme le projet n'est pas encore publié, je ne suis pas en mesure de fournir plus de détails et d'aperçus du site web.

Compétences sollicitées

  • Travail d'équipe : Planification du projet, réunions régulières
  • Gestion de projet : Méthodes Agile
  • Git : Versionning du site (historique des versions), collaboration en équipe
  • PHP : Backend global du site web, notamment avec le framework Symfony
  • SQL : Utilisation du SQL dans notre base de données MariaDB
  • JavaScript : Utilsé dans certaines pages pour récupérer du contenu ou modifier les agissements de la page.
  • TWIG : Structuration des pages
  • CSS/TAILWIND : Style des pages
