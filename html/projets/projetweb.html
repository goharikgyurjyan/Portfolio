Projet Web - ShoweRGB

ShoweRGB est le résultat du projet final des travaux dirigés de développement web. Il est le fruit des diverses notions apprises en cours, telles que l'architecture MVC, la gestion des données, les cookies, les sessions etc, le tout en PHP.

Pour ce projet de deux mois, nous avons dû travailler en trinômes ou quatuors afin de respecter un cahier de charge fourni par nos enseignants.
La consigne était de créer un site web marchand sur un thème au choix. Notre groupe a donc décidé d'opter pour une vente d'articles d'hygiène, avec une direction artistique colorée et décalée, rappelant les anciens sites internet.

Concernant l'organisation du projet, un répertoire git a été créé et nous nous sommes répartis le travail suivant les différentes fonctionnalités du projet. Le projet mêle le PHP, le HTML et le CSS. Pour ma part, je me suis chargée du CRUD (Create Read Update Delete) des produits et des différentes vues liées aux produits. En plus de ça j'ai également géré les messages flash ainsi que la vérification par email.

Compétences sollicitées

  • Travail d'équipe : Planification du projet, réunions régulières
  • Git : Versionning du site (historique des versions), collaboration en équipe
  • PHP : Création des vues, interactions avec la base de données, généricité du code...
  • HTML : Structuration des pages
  • CSS : Style des pages
