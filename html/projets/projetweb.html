Projet Web - ShoweRGB

ShoweRGB est mon projet final du cours de développement web (1er semestre 2024/2025). Il intègre les concepts fondamentaux appris en cours : architecture MVC, gestion des données, cookies, sessions, le tout en PHP

Ce projet de deux mois, réalisé en équipe de 3 ou 4, consistait à développer un site e-commerce selon un cahier des charges précis, mais avec un thème libre. Notre groupe a donc décidé d'opter pour une boutique d'articles d'hygiène, avec une direction artistique s'inspirant du design rétro du web pour un résultat coloré et décalé.

Concernant l'organisation du projet, un répertoire git a été créé et nous nous sommes répartis le travail suivant les différentes fonctionnalités du projet. Ce projet met en pratique le PHP, le HTML et le CSS, ainsi que quelques notions en SQL. Dans ce projet, je me suis chargée de l'implémentation des méthodes du CRUD (Create Read Update Delete) des produits et des différentes vues liées aux produits. En plus de ça j'ai également géré les messages flash grâce à la création d'une classe dédiée. Je me suis aussi occupée de la vérification par email en simulant l'envoi d'un email dans une fenêtre à part.

Compétences sollicitées

  • Travail d'équipe : Planification du projet, réunions régulières
  • Git : Versionning du site (historique des versions), collaboration en équipe
  • PHP : Création des vues, interactions avec la base de données, généricité du code...
  • HTML : Structuration des pages
  • CSS : Style des pages
  • SQL : Manipulation de bases de données, notamment avec PDO (PHP Data Object)
